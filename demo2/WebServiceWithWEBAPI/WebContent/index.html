Random API

Random values API, used for test mocks etc. Uses a custom random value web service and an external distance converter webservice.

Endpoints:

GET /distances

Returns a random distance in feet and metres.

Query parameters:

  • min [number] - inclusive minimum value in metres, default 0
  • max [number] - exclusive maximum value in metres, default 1000

Examples:

  • Random distance between 0 and 1000 metres: /distances
  • Random distance between 1 and 2 metres: /distances?min=1&max=2
