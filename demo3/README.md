# A simple analytics REST API

Deployed at https://analytics-rest.herokuapp.com.

## Dependencies

- Python 3
- pipenv
- MongoDB

## Setup

- `pipenv install`

## Run locally

- `python run.py` (or with auto-reloading etc: ``FLASK_DEBUG=1 FLASK_APP=run.py flask run`)

## Deploy

- `git subtree push --prefix demo3 heroku master`
